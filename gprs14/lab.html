Preinstalled GPy

If you have IPython and GPy installed on your machine you need to first start by typing 

  ipython notebook
into a terminal window (or the command window on Windows).

Then start a new notebook and cut and paste the following commands for the relvant lab class.

Lab 1

  import urllib
  urllib.urlretrieve('http://ml.dcs.shef.ac.uk/gpss/gprs14/lab1.ipynb', 'lab1.ipynb')
